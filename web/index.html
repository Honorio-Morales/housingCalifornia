Predictor de Viviendas en California

Llena el formulario con las características de un distrito y pulsa "Predecir".

Este ejemplo usa una API de regresión entrenada con el dataset California Housing. Ingresa los valores de las características y pulsa "Predecir" para enviar un request a la API. El servidor ejecuta el modelo y devuelve una estimación de MedHouseVal, que es el valor medio de la vivienda en el distrito en unidades de 100 000 USD (por ejemplo, 0.49 → aproximadamente 49 000 USD). También hay un botón "Simular" que genera valores aleatorios para ver distintos escenarios.
